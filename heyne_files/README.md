# Heyne et al. (2020) files used in this project
# _Heyne et al. 2020: Predicting Functional Effects of Missense Variants in Voltage-Gated Sodium and Calcium Channels_

This project adapts the machine-learning framework developed by Heyne et al. (2020) for predicting GOF vs LOF variants in voltage-gated ion channels.

The original code and datasets are available in the official repository:
https://github.com/heyhen/funNCion

Due to file size limitations and to preserve the original source, the files are not duplicated in this repository.

To reproduce the analysis, please download the following files from the original repository:

- featuretable4github_revision.txt
- predictiontable_all_variants_CACNA1SCN.txt
- pretty_featurenames2.txt
- R_functions4predicting_goflof_CACNA1SCN.R
- SupplementaryTable_S1_pathvariantsusedintraining_revision2.txt
- scncacaa_familyalignedCACNA1Acantranscript.txt

The scripts in this project expect these files to be available locally when running the pipeline.
